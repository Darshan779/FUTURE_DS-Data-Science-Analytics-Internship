## 📊 Dataset
Superstore Sales Dataset sourced from Kaggle (educational use).

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Analysis Performed
- Revenue Trend Analysis
- Top-Selling Product Identification
- Category-wise Performance Analysis
- Regional Sales Comparison

---

## 🎯 Conclusion
This project helped in understanding business performance metrics and generating actionable insights using data analytics.
